gauge.js

Native JavaScript gauge control

  • Download Source
  • Download Minimized
  • Plugin for JQuery

Ключевые особенности

  • Никаких картинок, спрайтов или CSS - только canvas
  • Никаких зависимостей. Jquery поддерживается, но совершенно не нужен
  • Гибкая система настроек
  • Любые значения шкалы в пределах до 10,000
  • Доступно и понятно стилизуем через настройки
  • Устанавливается через bower

Использование

Создаем пациента в HTML

<div id="canvas"></div>

И оживляем его!

var g = Gauge({
    id: "canvas",       // ID элемента для установки прибора
    width: 400,         // Размер
    maxValue: 6,        // Максимальное значение
    scaleRange: {
        warning: 4.5,   // начало стремной зоны
        danger: 5.5     // начало опасной зоны
    },
    value: 4            // текущее значение
});

Меняем значения

g.setValue(5);      // Установить новое значение в единицах, или
g.set(0.25);        // Установить новое значение в долях единицы

Примеры использования

Еще немножечко кастомизации

Gauge({
    id: "examples",
    width: 400,
    name: 'Спидометр',  // Имя прибора
    index: "км / ч",    // Единицы измерения
    scaleInside: true,  // Шкала не внесена внутрь
    maxValue: 250,
    colors: {           // Цвета:
        background: '#000', // — задника
        texts: '#fff',      // — цифр, букв и разметки
        hands: '#f00',      // — стрелки
        base: '#02fefe',    // — шкалы
        warning: '#fa0',    // — стремной зоны
        danger: '#f00'      // — опасной зоны
    },
    scaleRange: {
        warning: 200,
        danger: 235
    },
    value: 60
});

Gauge({
    id: "examples",
    width: 400,
    name: 'СДЕЛАНО В СССР',
    index: "V",
    maxValue: 1000
});

Поддержка браузерами

Прибор корректно отображается во всех современных браузерах*:

  • Chrome 6.0+
  • Safari 3.2+
  • Firefox 3.5+
  • IE 9
  • Opera 9.6+
  • Mobile Safari (iOS 3.0+)
  • Android 2.1+

*Чисто теоретически ;) Я не проверял
